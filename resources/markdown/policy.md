# Política de Privacidade

Bem-vindo ao sistema de prontuários. Este documento descreve como coletamos, usamos e protegemos suas informações pessoais.

## 1. Informações que Coletamos

Coletamos apenas informações estritamente necessárias para o funcionamento do sistema:

- **Dados do Paciente**: Nome, data de nascimento, informações de contato.
- **Dados do Atendimento**: Data, médico, especialidade, observações.
- **Dados do Usuário**: Acesso ao sistema (autenticação).

## 2. Uso das Informações

As informações são utilizadas exclusivamente para:
- Gerenciar o atendimento médico.
- Manter o histórico de consultas.
- Garantir a segurança e integridade dos dados.

## 3. Compartilhamento de Informações

Suas informações não são compartilhadas com terceiros, exceto quando:
- Exigido por lei.
- Necessário para a prestação do serviço (ex: integração com sistemas autorizados).

## 4. Segurança

Implementamos medidas de segurança robustas para proteger seus dados contra acesso não autorizado, alteração ou destruição.
